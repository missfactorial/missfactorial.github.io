Web Scraping and Automation
course coming Fall 2025!

Course Overview

Imagine needing market insights, price trends, or leads, but the data isn't readily available. How do you get it?

That's where web scraping comes in! Instead of spending hours manually copying and pasting data from the web, you'll learn how to automate data extraction from websites using programming techniques.

Who is this for? Whether you're a data scientist, analyst, researcher, or developer, this course will teach you how to efficiently automate web data collection. No prior web scraping experience required—just basic Python knowledge!

What You'll Learn:

  • Web Scraping Fundamentals – How websites work, HTTP, HTML, and APIs
  • BeautifulSoup & Selenium – Extracting data from static and dynamic pages
  • Scrapy – Building large-scale, high-performance web crawlers
  • Postman & APIs – Finding, testing, and interacting with APIs for data extraction
  • Handling Common Challenges – Proxies, rate limits, and user agents
  • AI-Assisted Web Scraping – Using AI tools to speed up development and troubleshooting
  • Automation & Deployment – Scheduling scrapers, cloud deployment, and maintaining data pipelines
